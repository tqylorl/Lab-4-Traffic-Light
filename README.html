ECE319K_Lab4

Readme
 
This is the starter project for ECE319K Lab 4. For more information about Lab 4, see Lab 4. Traffic Light. When running this project, the USB cable must be attached to the PC and a terminal program (Putty or CCS terminal) must be opened with a baud rate of 115200 bits/sec.
 

Hardware Required

  • MSPM0G3507 LaunchPad™ Development Kit (LP-MSPM0G3507)
  • Three B3F switches Datasheet
  • Six HLMP-4700 LEDs Datasheet
  • Three 10k and six 470ohm resistors

Pin assignments
  • Add your EID and run the code to see to which pins you should connect the switch and LED
  • Serial TxD: PA10 is UART0 Tx (MSPM0 to PC) index 20 in IOMUX PINCM table
  • Serial RxD: PA11 is UART0 Rx (PC to MSPM0) index 21 in IOMUX PINCM table
LaunchPad Jumpers
  • Remove J16 J17 J18: Disconnects light sensor from the microcontroller PA22 PA27 PA26
  • J21: Connects PA10 to XDS_UART
  • J22: Connects PA11 to XDS_UART
